# Práctica con CSS grid básica

Realiza en codepen o en tu IDE (VS code) unas cuadrículas iguales que la de la imágen adjunta (practica-basica-grid.png).

Estructura del html, el css tienes que hacerlo tú.

```html
<article>
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
  <div>5</div>
  <div>6</div>
  <div>7</div>
</article>
```
