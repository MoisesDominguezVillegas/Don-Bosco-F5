Responsive Web Desing

Usando la propiedad width

Si la propiedad width en CSS está definida al 100%, la imagen será "responsive" y se reescalará

Código del ejemplo
<img src="assets/img/img_girl.jpg" style="width:100%;">

Usando la propiedad max-width

Si la propiedad max-width está definida al 100%, la imagen será "responsive" y se reescalará, pero nunca por encima de su tamaño original.

Código del ejemplo
<img src="assets/img/img_girl.jpg" style="max-width:100%;height:auto;">

Mostrar diferentes imágenes dependiendo del ancho del navegador

El elemento HTML "picture" permite definir diferentes imágenes dependiendo del tamaño del navegador.

Redimensiona el navegador para ver cómo cambia la imagen de abajo en función del tamaño:

Código del ejemplo
<picture> <source srcset="assets/img/img_smallflower.jpg" media="(max-width: 600px)"> <source srcset="assets/img/img_flowers.jpg" media="(max-width: 1500px)"> <source srcset="assets/img/flowers.jpg"> <img src="assets/img/img_smallflower.jpg" alt="Flowers"> </picture>

Tamaño del texto adaptable

El tamaño de la fuente de texto puede ser modificado mediante la medida "vw", que significa "viewport width" o ancho del display (ancho de la pantalla o directamente de google, ancho de la ventana gráfica).

De esta manera el tamaño de la fuente se modificará en función del tamaño del navegador.

Hello World

Resize the browser window to see how the text size scales.

Código del ejemplo
<h1 style="font-size:10vw">Hello World</h1> <p style="font-size:2vw;">Resize the browser window to see how the text size scales.</p>

Media Queries

Además de redimensionar el texto y las imágenes, es muy común utilizar media queries en las páginas web responsive o adaptativas.

Con las media queries puedes definir estilos totalmente diferentes para los todos tamaños del navegador.

Ejemplo: Redimensiona el tamaño del navegador para ver cómo los tres elementos div se muestran horizontalmente en grandes dimensiones de pantalla, o apilados verticalmente en dispositivos pequeños.

<style> .left, .right { float: left; width: 20%; /* The width is 20%, by default */ } .main { float: left; width: 60%; /* The width is 60%, by default */ } /* Use a media query to add a breakpoint at 800px: */ @media screen and (max-width: 800px) { .left, .main, .right { width: 100%; /* The width is 100%, when the viewport is 800px or smaller */ } } </style> <div> <div class="left"> <div class="w3-blue w3-padding">Left Menu<br><br><br></div> </div> <div class="main"> <div class="w3-light-gray w3-padding">Main Content<br><br><br></div> </div> <div class="right"> <div class="w3-green w3-padding">Right Content<br><br><br></div> </div> </div>
