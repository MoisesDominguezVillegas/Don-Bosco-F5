Vamos a trabajar con funciones

Declarando funciones

A menudo necesitamos realizar una acción similar en muchos lugares de JavaScript, cómo mostrar un mensaje, conectar con un servicio, etc. Por ello para no repetir el mismo código (algo contrario a cualquier norma o estandar de programación) se crean las funciones. Que son bloques de código que permiten ser llamadas para evitar tener que repetirlos. 

function nombreFuncion(params){
   return params;
}

nombreFuncion('Hola coder');

Funciones Anónimas

A menudo no necesitamos darle un nombre a la función, ya que no va a ser llamada si no que se ejecutará en respuesta a un evento o una acción

function (params) {
   return params;
}

Estas funciones reciben el nombre de anónimas. A veces se las llama "callbacks" ya que se producen en respuesta a un servicio, cómo la llamada a una API.

Cómo la función que controla la carga del JavaScript para hacerla después del HTML.

windows.onload = function () {
   return "Página cargada";
};

Si escuchásemos el evento 'click' de un botón (por ejemplo) declararemos una función anónima que reaccione a dicho evento 

btn.addEventListener('click', function() {
   return "Has pulsado el botón de enviar";
});

Expresiones de funciones

Existen varias maneras de declarar una función con nombre, es decir, que no sea anónima. Tenemos la versión normal, conocida como declaración de funciones.

function nombreFuncion(params){
   return params;
};

La expresión de funciones es un concepto nuevo, ya que la función no es una "estructura mágica del lenguaje" si no un tipo de datos especial. Por lo que podemos guardar una función en una variable. Sería del siguiente modo: 

let nombreVariable = function(params) {
   return params;
};
nombreVariable('Hola coders');

Las expresiones de funciones deben terminar con un ';'. A diferencia de los bloques 'for', 'if' y 'function f { }' que pueden no llevarlos, ya que están bien definidos desde el inicio hasta el final. Las expresiones de funciones al ser un valor asignado a una variable, debe ser delimitado con un ";".
